# 🌌 Luminous Sanctuary – AI Powered Emotional & Learning Support System

## Problem Statement

Students and individuals often struggle with emotional stress, lack of focus, and unstructured learning habits. Existing platforms either focus only on productivity or only on mental well-being — not both. There is a need for a unified system that:

* Understands user emotional state
* Provides AI-driven support
* Helps with structured learning
* Tracks progress
* Works without complex setup

---

## Solution

**Luminous Sanctuary** is an AI-powered support system that combines emotional wellness + learning productivity into a single platform.
It provides adaptive guidance, structured learning tools, and AI-based interaction to help users stay focused and mentally balanced.

The system uses Firebase + React + TypeScript to deliver a fast and scalable experience.

## How to Run

### 1. Clone Repository

```bash
git clone https://github.com/pareesa969/Luminescent-Sanctuary.git
```

### 2. Navigate to Project

```bash
cd luminous-sanctuary
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Add Environment Variables

Create `.env`

```
VITE_FIREBASE_API_KEY=
VITE_FIREBASE_AUTH_DOMAIN=
VITE_FIREBASE_PROJECT_ID=
VITE_FIREBASE_STORAGE_BUCKET=
VITE_FIREBASE_MESSAGING_SENDER_ID=
VITE_FIREBASE_APP_ID=
```

### 5. Run Development Server

```bash
npm run dev
```

### 6. Build Production

```bash
npm run build
```
